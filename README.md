# README

### Prerequisites
* Ruby version 2.4.1

* Rails version 5.1.3

### How to view application

```
bundle install
```
```
rails db:migrate
```
```
rails db:seed
```
```
rails server
```

### Routes to view admin
* '/admin' will lead you to the admin dashboard. User accounts are not created.

### How to run the test suite
```
bundle exec rspec
```
